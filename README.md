## 🛠️ Troubleshooting – le mode *panique*
| Problème | Fix en 10s |
|---|---|
| **Rien ne se passe** | Installe [VC++ Redist](https://aka.ms/vs/17/release/vc_redist.x64.exe) |
| **Windows Defender** | Propriétés → Débloquer → OK |
| **DLL manquant** | Même lien VC++ ci-dessus. |
