Si l'application ne se lance pas, installez dans l'ordre :

1. **Microsoft Visual C++ Redistributable 2019-2022** (x64)
   - 🔗 [Télécharger ici](https://aka.ms/vs/17/release/vc_redist.x64.exe)
   
2. **.NET Framework 4.8** ou supérieur
   - 🔗 [Télécharger ici](https://dotnet.microsoft.com/download/dotnet-framework/net48)

#### 💻 Configuration Minimale
- **OS** : Windows 10/11 (64-bit recommandé)
- **RAM** : 2 GB minimum, 4 GB recommandé
- **Espace** : 100 MB libres
- **Processeur** : Dual Core 1.6 GHz

### 🛡️ Sécurité & Antivirus

**⚠️ Windows Defender / Antivirus :**
- L'application peut être signalée comme "non reconnue"
- C'est **NORMAL** pour les applications non signées numériquement
- **Solution** : Cliquez sur "Plus d'informations" → "Exécuter quand même"
- **Alternative** : Ajoutez le dossier aux exclusions de votre antivirus

## 🎯 Guide d'Utilisation

### ➕ Ajouter un Nouveau Favori

1. **Cliquez** sur le bouton rose **"+ AJOUTER UN NOUVEAU FAVORI"**
2. **Renseignez les informations :**
   - 🏷️ **Nom du favori** : Nom qui s'affichera (ex: "YouTube", "Steam")
   - 🔗 **URL/Chemin** : 
     - Sites web : `https://youtube.com`
     - Applications : `C:\Program Files\Steam\steam.exe`
     - Dossiers : `C:\Users\VotreNom\Documents`
     - Utilisez **BROWSE** pour sélectionner facilement
3. **Choisissez le type :**
   - 🌐 **Site Web** - Pour les URLs internet
   - 🎮 **Jeu/Game** - Pour vos jeux favoris
   - 📁 **Dossier** - Pour les répertoires
   - 📱 **Application** - Pour les logiciels
4. **Cliquez** sur **"AJOUTER LE FAVORI"**

### 🔍 Gérer vos Favoris

#### Recherche Intelligente
- Utilisez la barre **"Rechercher dans tes favoris..."**
- Tapez n'importe quel mot-clé
- Les résultats s'affichent en temps réel

#### Actions Disponibles
- **📂 Ouvrir** : Lance votre favori (site/app/dossier)
- **✏️ Éditer** : Modifie les informations du favori
- **🗑️ Supprimer** : Efface définitivement le favori

#### Navigation par Catégories
- **Filtrez** par type dans la liste de droite
- **Voyez** les statistiques en temps réel en haut
- **Organisez** vos favoris par usage

## 🆘 Résolution de Problèmes

### ❌ L'application ne se lance pas
1. **Vérifiez** que vous avez installé Visual C++ Redistributable
2. **Téléchargez** le package complet `.zip` au lieu du `.exe` seul
3. **Désactivez** temporairement votre antivirus
4. **Exécutez** en tant qu'administrateur (clic droit → "Exécuter en tant qu'administrateur")

### ❌ Erreur "DLL manquante"
1. **Installez** .NET Framework 4.8
2. **Redémarrez** votre ordinateur
3. **Re-téléchargez** l'application

### ❌ Les favoris ne s'ouvrent pas
1. **Vérifiez** que les chemins sont corrects
2. **Pour les sites web** : assurez-vous d'inclure `https://`
3. **Pour les applications** : vérifiez que le fichier .exe existe
